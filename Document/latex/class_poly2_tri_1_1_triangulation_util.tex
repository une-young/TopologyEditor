This method detects if two line segments (or lines) intersect, and, if so, the point of intersection. Use the {\itshape first\+Is\+Segment}  and {\itshape second\+Is\+Segment}  parameters to set whether the intersection point must be on the first and second line segments. Setting these both to true means you are doing a line-\/segment to line-\/segment intersection. Setting one of them to true means you are doing a line to line-\/segment intersection test, and so on. Note\+: If two line segments are coincident, then no intersection is detected (there are actually infinite intersection points).
