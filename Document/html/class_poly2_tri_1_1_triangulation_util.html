This method detects if two line segments (or lines) intersect, and, if so, the point of intersection. Use the firstIsSegment and secondIsSegment parameters to set whether the intersection point must be on the first and second line segments. Setting these both to true means you are doing a line-segment to line-segment intersection. Setting one of them to true means you are doing a line to line-segment intersection test, and so on. Note: If two line segments are coincident, then no intersection is detected (there are actually infinite intersection points).

매개변수
ptStart0The first point of the first line segment.
ptEnd0The second point of the first line segment.
ptStart1The first point of the second line segment.
ptEnd1The second point of the second line segment.
firstIsSegmentSet this to true to require that the intersection point be on the first line segment.
secondIsSegmentSet this to true to require that the intersection point be on the second line segment.
coincidentEndPointCollisionsSet this to true to enable collisions if the line segments share an endpoint
pIntersectionPtThis is set to the intersection point if an intersection is detected.
반환값
True if an intersection is detected, false otherwise.
